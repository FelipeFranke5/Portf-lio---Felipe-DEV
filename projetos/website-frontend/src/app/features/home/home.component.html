O que eu aprendi na prática

Já realizei atividades que envolvem instalar e realizar o deploy de aplicações web que possuem um Back-end e se comunicam com sistemas externos. Em especial, menciono o Apache Tomcat para Java e Uvicorn para Python.

Host na AWS
Deploy de uma aplicação em uma instância EC2 da AWS como parte de um projeto pessoal. Desde a configuração inicial no painel administrativo, até definir quais comandos são executados durante a inicialização de um servidor. No meu trabalho do dia a dia, ainda não tive a oportunidade de instalar e configurar uma instância do zero, porém já contribui sendo um analista de sustentação provendo troubleshooting e auxiliando os administradores com a manutenção de servidores já em funcionamento.

Acompanhei de perto um processo de migração de uma aplicação que antes estava hospedada no on-premises (infra do cliente) para a AWS e foi um desafio bem interessante. Os projetos de migração são aqueles que colocam "pra jogo" tudo o que você sabe sobre trabalho em equipe, pois em um ambiente com uma infra-estrutura grande, um bom planejamento e comunicação com todos os times responsáveis são importantes (ou como gostam de falar por ai, mas não falo muito: imprescindíveis).
